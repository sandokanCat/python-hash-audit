# Security Policy

We take the security of this project seriously. If you discover a vulnerability, please follow the steps below.

## Supported Versions

Currently, only the latest version (`main`) receives critical security updates.

| Version | Supported |
| :--- | :--- |
| `1.x` | ✅ |
| `< 1.0` | ❌ |

## Reporting a Vulnerability

Please do **NOT** report security vulnerabilities via public GitHub Issues.

Instead, please send an email to [dev@sandokan.cat](mailto:dev@sandokan.cat). Try to include as much information as possible to reproduce the issue.

You will receive a response within a reasonable timeframe (usually 48h) acknowledging receipt.
